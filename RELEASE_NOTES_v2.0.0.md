# NexusOS v2.0.0

Don't trust. Verify.

## Feature Highlights

- Coding Agent: scans codebases, analyzes architecture, writes code, runs tests, and iterates on failures.
- Screen Poster Agent: human-like social workflow automation with mandatory human approval before posting.
- Web Builder Agent: natural language to deployable React/Tailwind sites with Three.js scene generation.
- Workflow Studio: visual DAG automation with retries, parallel execution, and checkpoints.
- Self-Improve Agent: performance tracking, strategy learning, prompt optimization, and rollback-capable improvements.
- Designer Agent: UI generation, component library synthesis, screenshot-to-code analysis, and token generation.
- Desktop UI refresh: neural background, holographic panels, richer visual telemetry, and upgraded interactions.
