How it works

One governed loop from intent to execution.

The system translates a human goal into a governed action chain. Every step is observable, measurable, interruptible, and auditable.

1

User sends a goal

Chat, schedule, API, or an app event gives the system a clear objective to pursue.

2

Agent plans with cognitive loop

Perceive → Reason → Plan → Act → Reflect → Learn, not one-shot prompting.

3

Governance checks every action

Fuel metering, capability checks, path rules, egress rules, and audit obligations run before execution.

4

Human approves if needed

Purchases, sensitive changes, or high-risk acts route into a real HITL approval experience.

5

Agent executes governed actions

File, shell, web, browser, Docker, API, and self-modify actuators stay behind the same kernel guardrails.

6

Agent learns and evolves

Successful patterns move into memory, strategy scores update, and L4+ agents improve themselves over time.
