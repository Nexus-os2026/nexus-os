# NEXUS OS

**Tagline:** Don't trust. Verify.

NEXUS OS is a consumer-facing governed agent operating system built to run policy-bounded agents on top of NEX Lang.

## Architecture

```text
+---------+
| Host OS |
+---------+
     |
     v
+----------+
| NEX Lang |
+----------+
     |
     v
+----------+
| NEXUS OS |
+----------+
     |
     v
+--------+
| Agents |
+--------+
```
